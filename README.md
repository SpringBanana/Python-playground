# Python HW and projects

Этот репозиторий собрал мои учебные домашние задания и **сторонние** проекты, которые я делаю в процессе изучения Data Science, Python и сопутствующих технологий.

## Структура репозитория

- [`Python_HW`](./Python_HW) – папка с домашними заданиями по Python (курсы, учебные модули и практические задачи).
- [`Projects`](./Projects) – небольшие сайд-проекты, эксперименты и прототипы.
- [`Notebooks`](./Notebooks) – Jupyter-ноутбуки с анализом данных, визуализациями и экспериментами с моделями.

## Используемый стек

- Python (pandas, NumPy, matplotlib, seaborn, scikit-learn и др.).
- SQL для работы с базами данных и аналитическими запросами.
- Git и GitHub для версионирования кода и хранения проектов.
