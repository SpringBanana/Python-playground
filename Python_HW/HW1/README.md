# HW1 - Основы Python

Первая домашняя работа посвящена базовым конструкциям языка Python и работе с условиями, вводом данных и простыми вычислениями.

## Цели домашней работы

- Закрепить работу со строками и числами.
- Научиться использовать условные операторы.
- Потренировать работу с пользовательским вводом.
- Освоить базовую логику ветвлений на практических задачах.

---

## Задание 1. Сравнение длин двух фраз

Даны две текстовые переменные с фразами произвольной длины.  
Необходимо написать программу, которая:

- Сравнивает длину первой и второй фразы.
- Выводит, какая фраза длиннее: первая или вторая.
- Если длины совпадают, выводит сообщение о равной длине фраз.

Ожидаемый результат работы программы — текстовое сообщение о том, какая фраза длиннее, либо что фразы равной длины.

---

## Задание 2. Определение типа года

Дана переменная с числом, обозначающим год.  
Необходимо написать программу, которая:

- Определяет, является ли указанный год високосным или обычным.
- Выводит одно из двух сообщений: «Високосный год» или «Обычный год».

Программа должна корректно обрабатывать разные значения года.

---

## Задание 3. Определение знака зодиака

Программа должна запросить у пользователя:

- День рождения.
- Месяц рождения (в текстовом виде).

На основе введенной даты необходимо:

- Определить знак зодиака пользователя.
- Вывести сообщение вида: «Ваш знак зодиака: <знак>».

Задача предполагает использование ветвлений для проверки диапазонов дат.

---

## Задание 4. Подбор упаковки по размерам

Даны размеры товара в сантиметрах: ширина, длина и высота.  
Необходимо написать программу, которая выбирает подходящий тип упаковки по следующим правилам:

- Если каждое из трех измерений меньше либо равно 15 см, выводится «Коробка №1».
- Если хотя бы одно измерение больше 200 см, выводится «Упаковка для лыж».
- Если хотя бы одно измерение больше 15 см, но меньше 50 см, выводится «Коробка №2».
- Во всех остальных случаях выводится «Коробка №3».

Результатом работы программы является текст с названием выбранной упаковки.

## Задание 5. "Счастливый" билет

Дано шестизначное число, обозначающее номер проездного билета.  
Необходимо написать программу, которая:

- Разделяет номер на две части: первые три цифры и последние три цифры.
- Считает сумму цифр в каждой части.
- Сравнивает суммы и выводит:
  - «Счастливый билет», если суммы совпадают.
  - «Несчастливый билет», если суммы различаются.

Задание направлено на тренировку работы с числами и индексами/преобразованием типов.

## Задание 6. Площади фигур

Программа должна уметь вычислять площади трех типов фигур:

- круг;
- треугольник;
- прямоугольник.

Алгоритм работы:

1. Запросить у пользователя тип фигуры (круг, треугольник, прямоугольник).
2. В зависимости от выбранного типа:
   - Для круга запросить радиус и вычислить площадь круга.
   - Для треугольника запросить длины трех сторон и вычислить площадь (с использованием соответствующей формулы).
   - Для прямоугольника запросить длины двух сторон и вычислить площадь.

В результате программа выводит сообщение с площадью выбранной фигуры.
