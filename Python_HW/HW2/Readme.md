## Задание 1 — Средняя буква в слове

Программа получает слово из латинских букв и:

- определяет длину строки;
- если длина **нечётная** — выводит одну среднюю букву;
- если длина **чётная** — выводит две средние буквы.

**Примеры:**

```python
word = 'test'
# Результат:
es

word = 'testing'
# Результат:
t
```

## Задание 2 — Сумма до первого нуля

Напишите программу, которая последовательно запрашивает у пользователя числа (по одному за раз) и после первого нуля выводит сумму всех ранее введённых чисел.

**Примеры работы программы:**

1.

```text
Введите число:
1
Введите число:
4
Введите число:
6
Введите число:
0

Результат:
11
```
2.

```text
Введите число:
0

Результат:
0
```
## Задание 3 — Идеальные пары для dating-сервиса

Мы делаем [MVP](https://ru.wikipedia.org/wiki/%D0%9C%D0%B8%D0%BD%D0%B8%D0%BC%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE_%D0%B6%D0%B8%D0%B7%D0%BD%D0%B5%D1%81%D0%BF%D0%BE%D1%81%D0%BE%D0%B1%D0%BD%D1%8B%D0%B9_%D0%BF%D1%80%D0%BE%D0%B4%D1%83%D0%BA%D1%82) dating-сервиса, и у нас есть список парней и девушек.

Но мы не будем никого знакомить, если кто-то может остаться без пары.

```python
boys = ['Peter', 'Alex', 'John', 'Arthur', 'Richard']
```

```text
Внимание, кто-то может остаться без пары!
```

## Задание 4 — Средняя температура в странах

У нас есть список, содержащий информацию о среднедневной температуре в Фаренгейтах за произвольный период по странам (структура данных в примере).  
Необходимо написать код, который рассчитает среднюю температуру за период в **Цельсиях** для каждой страны.

**Пример работы программы:**

```python
countries_temperature = [
    ['Таиланд', [75.2, 77, 78.8, 73.4, 68, 75.2, 77]],
    ['Германия', [57.2, 55.4, 59, 59, 53.6]],
    ['Россия', [35.6, 37.4, 39.2, 41, 42.8, 39.2, 35.6]],
    ['Польша', [50, 50, 53.6, 57.2, 55.4, 55.4]]
]
```
Результат:
```text
Средняя температура в странах:
Таиланд  -  23.9 C
Германия  -  13.8 C
Россия   -  3.7 C
Польша   -  12.0 C
```
## Задание 5  — Проверка транспортных номеров

Имеется список с транспортными номерами.  
Необходимо написать код, который проверит каждый номер на валидность по шаблону:

- 1 буква;
- 3 цифры;
- 2 буквы;
- 2–3 цифры.

Важно: не все буквы кириллического алфавита используются в транспортных номерах.

Если номер валиден — нужно вывести его в нужном формате (см. пример).  
Если номер невалиден — вывести текст об ошибке.  
При решении рекомендуется использовать регулярные выражения.

**Пример работы программы:**

```python
car_ids = ['А222ВС96', 'АБ22ВВ193']
```
Результат:
```text
Номер А222ВС валиден. Регион: 96
Номер АБ22ВВ193 не валиден
```
## Задание 6 — Подсчет средних просмотров уникальных пользователей
Дан поток логов по количеству просмотренных страниц для каждого пользователя (пользователь,дата;просмотры). 
Вам необходимо написать алгоритм, который считает среднее значение просмотров на пользователя. 
Т. е. надо посчитать отношение суммы всех просмотров к количеству уникальных пользователей. 
И тут регулярные выражения облегчат немного реализацию

**Пример работы программы:**

1.
```python
stream = [
    'user4,2021-01-01;3',
    'user3,2022-01-07;4',
    'user2,2022-03-29;1',
    'user1,2020-04-04;13',
    'user2,2022-01-05;7',
    'user1,2021-06-14;4',
    'user3,2022-07-02;10',
    'user4,2021-03-21;19',
    'user4,2022-03-22;4',
    'user4,2022-04-22;8',
    'user4,2021-05-03;9',
    'user4,2022-05-11;11'
]

```
Результат:
```text
Среднее количество просмотров на уникального пользователя: 23.25
```

2.
```python
stream = [
    'user100,2022-01-01;150',
    'user99,2022-01-07;205',
    'user1001,2022-03-29;81'
]
```
Результат:
```text

```
